Three kinds of responsibilities

  • Introduction
  • StaticTyping
  • Unit testing
Any class should do one of these three responsibilities:
  • Edge objects, objects that connect to other processes;
  • Housekeeping objects, objects that provided internal infrastructure;
  • Domain objects, objects that represent domain entities
